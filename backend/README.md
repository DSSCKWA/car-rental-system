# Car Rental System backend

Create venv:

```powershell
python -m venv venv
```

Activate venv:

```powershell
Set-ExecutionPolicy -ExecutionPolicy Bypass -Scope Process -Force; .\venv\Scripts\Activate.ps1
```

Install deps:
```powershell
pip install -r requirements.txt
```

Run app:
```powershell
flask run
```
